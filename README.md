# Kute

Kute is a tiny, cross-platform and CPU-only graphics library in C. I use it to learn the very fundamentals of graphics.
The goal is to have something very simple, easy to understand, and that can be improved. Any suggestion is vastly appreciated.

## Examples

Check the **examples** folder to see how to integrate kute in various platforms (web and Windows targets are supported for now).

## Web

Kute is easily integrated in a web page. You can use the template in the **web** folder. Essentially you just need to instantiate the wasm module
in which you use kute and you're good to go.

## References

It is greatly inspired by the amazing work done in **olive.c** (check it out at https://github.com/tsoding/olive.c).
